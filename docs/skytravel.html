SkyTravel is the part of the AlpacaPi suite of software. It allows full control of an Alpaca based observatory. Specifically you can control the dome, cameras, focusers, telescope mount, switches, all from the same program. It does NOT REQUIRE that you use SkyTravel for all of these devices.

SkyTravel implements all of the client interfaces described in the Client Apps section. It uses the Alpaca Discovery protocol to find all of the devices on your network that speak Alpaca and then gives you quick access to them.

AAVSO Target Tool

AAVSO provides a list of objects of interest via a system called "Target tool". This list shows observations that professional astronomers would like assistance with in the form of additional observations, many of which can be accomplished by amateur astronomers.

In order to use the target tool, you must create an account.


The yellow text objects are the alerts from the target tool


When zoomed in, the alert number appears as well.
